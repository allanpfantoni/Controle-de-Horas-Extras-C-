Título: Controle de Horas Extras (C)

Descrição:
Projeto da aula de Programação de Computadores, 2º período. Sistema simples em C para cadastrar funcionários, registrar horas extras, aprovar como supervisor ou gestor e gerar relatórios de horas aprovadas.

Como usar:
1- Cadastre os funcionários no sistema.
2- Registre as horas extras de cada funcionário.
3-Supervisor aprova ou rejeita as horas.
4-Gestor aprova definitivamente.
5-Confira os relatórios para ver as horas aprovadas.
(Dica: No Replit, cole o código no main.c e clique em Run. Siga o menu no console.)

Menu do programa:
1: Cadastrar funcionário
2: Listar funcionários
3: Registrar horas extras
4: Aprovação supervisor
5: Aprovação gestor
6: Relatório de horas extras
0: Sair
